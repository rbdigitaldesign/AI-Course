1. List content to be completed

In the table below, please document content that is unable to be completed within the course development cycle. Please provide details of any Teaching and Learning Innovation (TLI) related support currently underway (e.g., course development, media production, or library support). The completion timeline for these tasks should be in readiness for the start of teaching. Your ATP/CQP member may assist you in determining this.

The following are some suggestions of questions to consider. If any of the following is true, please also include them in the table below:

  • Are there any trigger warnings required?
  • Is there any group work that may need updating/revising by the Course Coordinator?
  • Are there any student-facing resources that may need updating (H5Ps, quizzes)? 
  • Are there any media resources that need rebranding/sending to the media team?
  • Has the reading list been completed in Leganto?
  • Have there been discussion boards created for students to use? (e.g. assessment, end of module)
  • Are there any legislative frameworks/external factors that require updates prior to teaching?
